# 💻 Sakshi's Developer Portfolio

Welcome to my personal portfolio — a showcase of my technical journey, projects, and design approach. Built with passion, sprinkled with animations, and driven by clean, efficient code.

## 🚀 Live Site
(https://portfoliosakshii.netlify.app/)
## 🛠️ Tech Stack

- **Frontend:** React, HTML5, CSS3, JavaScript, reactjs
- **Animations & Styling:** Framer Motion, Tailwind CSS / Styled Components
- **Deployment:** Netlify
- **Version Control:** Git, GitHub

## 📂 Features

- Beautifully responsive UI with subtle motion effects
- Projects section highlighting full-stack and algorithmic work
- About section blending code and creativity
- Clean structure with reusable components


## 🔧 Setup Instructions

```bash
# Clone the repository
git clone https://github.com/sakshi0444/Portfolio-Sakshi.git

# Navigate into the project directory
cd Portfolio-Sakshi

# Install dependencies
npm install

# Run the dev server
npm start
